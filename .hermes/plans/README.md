# Hermes plans

Plan-mode artifacts produced by [Hermes Agent](https://hermes-agent.nousresearch.com/) live here.

These are working documents — proposals, audits, implementation plans — written before code changes land. They are tracked in git so the reasoning behind a change is preserved alongside the change itself.

Conventions:

- Single-file plan: `YYYY-MM-DD_HHMMSS-<slug>.md`
- Multi-file plan: `YYYY-MM-DD-<slug>/00-INDEX.md` plus dimension files in the same directory.

Safe to delete after the corresponding work has merged and the lessons have moved into `AGENTS.md`, a SKILL.md, or `memory.md`.
