# IMO HQ (@IMOHQ)
Official Twitter account for the International Maritime Organization (IMO), the UN specialized agency responsible for maritime safety, security, and environmental protection. Provides announcements on maritime security incidents, piracy alerts, shipping regulations, port state control, and international maritime cooperation. Critical source for monitoring maritime security threats, shipping disruptions, and naval incidents affecting global trade.
Keywords: piracy, attack, hijacking, security incident, maritime threat, Red Sea, Gulf of Aden, Strait of Hormuz, South China Sea, Malacca Strait, seizure, boarding, Houthi, naval incident
